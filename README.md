# AI-Agent应急响应系统--AI-Agent团队建设


## 项目简介

本项目是一个基于AI的应急响应系统，使用CrewAI框架构建了一个智能安全分析团队，能够自动执行系统监控、进程分析、日志审计、威胁检测和应急响应等任务。系统提供了图形用户界面(GUI)，使用户能够直观地操作和查看分析结果。其实不局限于安全方面，愿景通过自动生成AI和工具是解决用户的任何任务

## 系统特点

- **智能分析**：利用多个AI Agent协同工作，自动分析系统安全状态
- **可视化界面**：提供直观的图形界面，易于操作和查看结果
- **实时监控**：实时监控系统进程、服务和日志
- **威胁检测**：自动检测潜在的安全威胁
- **应急响应**：提供系统隔离和恢复功能

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动图形界面

```bash
python gui_main.py
```

### 使用命令行界面

```bash
python main.py
```

## 功能说明

### 1. 系统监控

监控系统进程、服务和资源使用情况，定期收集数据并生成报告。

### 2. 进程分析

分析系统中运行的进程，识别可疑或异常的进程。

### 3. 日志审计

收集和分析系统日志，识别潜在的安全事件。

### 4. 威胁检测

综合分析系统状态，检测潜在的安全威胁。

### 5. 应急响应

提供系统隔离和恢复功能，应对安全事件。

## 界面说明

系统界面分为两部分：

1. **开始界面**：展示流动多边形动画，点击任意处进入主界面
2. **主界面**：
   - 左侧功能区：提供功能选择
   - 右侧内容区：显示功能操作界面和结果

## 配置说明

系统配置文件位于`config.py`，可以根据需要修改以下配置：

- API密钥
- 模型配置
- 监控间隔时间
- 日志和配置文件路径

## 注意事项

- 部分功能需要管理员权限才能正常运行
- 确保系统已安装Python 3.8或更高版本(我的是Python 3.11.12)

- 使用前请确保已正确配置API密钥
