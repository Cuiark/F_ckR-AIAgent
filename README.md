## 使用方法：

#### 1、搭建环境：

`python >= 3.12.4` (我的环境为3.12.4)

`pip install requirements.txt`
#### 2、设置DeepSeek/OpenAi的API-Key（由于地区问题，只测试了DeepSeek）:

**config\constants.py**中的：

`OPENAI_API_KEY = "NULL"`

`DEEPSEEK_API_KEY = ""`

`DEEPSEEK_API_BASE = "https://api.deepseek.com/v1"`



#### 3、命令行版运行：

运行 `python main.py`



#### 4、 ui版运行：

运行 `pyhon gui_main.py`
